This is the project of Guy Eshel
MMN 14 - assembler in c

please run this project on Linux
make sure you have gcc installed
compile using make, the output program is 'assembler'

running instructions:

In the terminal, enter 
./assembler <paths of .as files separated by spaces> (enter the names without .as)
 the program will compile each code and will create 3 new files in the same directory in case there are no syntax errors.(.ext,.ob,.ent)
 if there are errors they will be printed to stdout.
